# Contributing to DjangoPydanticBridge

Thank you for your interest in contributing to DrfPydanticBridge! Here's how you can contribute:

1. **Fork and Clone**: Fork the repository on GitHub and clone it locally to your machine.
2. **Set Up**: Install the required development dependencies and set up the development environment.
3. **Branch**: Create a new branch for your changes.
4. **Make Changes**: Make your changes and commit them with a meaningful commit message.
5. **Test**: Ensure all tests pass locally.
6. **Push**: Push your changes to your fork on GitHub.
7. **Pull Request**: Open a pull request from your fork to the main repository.

Please ensure you follow the code style guidelines and add tests for any new functionality.
